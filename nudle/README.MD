Association & collection Mapping
-------------

Chain방식의 경우 <br>
1 => M <= 1 이 형태에서는 M만 양쪽의 데이터를 참조할 수 있다. <br>
[ 한쪽에서 모든 데이터를 가져 올 수 가 없다. ]
<br>
1 <= M (1) <= M 이 형태 혹은 1 => M (1) => M <br>
collection 으로만 진행 하던지 혹은 association으로만 진행 하던지 <br>
둘 중 하나로만 진행해야 코드 성공함
